The react-hot-toast library is designed to display toast notifications in a fast and lightweight manner, but the duration of the toast is controlled by the duration property which is set to 5000ms (5 seconds) by default.

javascript
import { toast } from 'react-hot-toast';

toast('Hello, world!', {
  duration: 8000 // 8 seconds
});

In addition to duration, there are other properties you can use to customize the behavior of the toast notification, such as the position property to set the location of the toast, and the style property to apply custom CSS styles to the toast.

Keep in mind that toast notifications should be used sparingly and should not distract users from the main content of your application. If your toast notifications are appearing and disappearing too quickly, consider whether they are necessary or whether they can be displayed in a less intrusive manner.

You can use the promise function of react-hot-toast to control the duration of the toast notification based on the outcome of a Promise. Here's an example of how you can use it to display a success message for 5 seconds after a user successfully registers:

javascript
import { useState } from 'react';
import { toast } from 'react-hot-toast';

function RegisterPage() {
  const [isRegistering, setIsRegistering] = useState(false);

  const handleRegister = async () => {
    setIsRegistering(true);

    // make API call to register user
    const response = await fetch('/api/register', {
      method: 'POST',
      body: JSON.stringify({ /* registration data */ }),
    });

    if (response.ok) {
      // registration succeeded
      toast.promise(
        Promise.resolve('Registration succeeded'),
        {
          loading: 'Please wait...',
          success: 'Registration succeeded!',
          duration: 5000,
        }
      );
    } else {
      // registration failed
      toast.error('Registration failed');
    }

    setIsRegistering(false);
  };

  return (
    <div>
      <h1>Register</h1>
      {isRegistering ? (
        <div>Please wait...</div>
      ) : (
        <button onClick={handleRegister}>Register</button>
      )}
    </div>
  );
}

In this example, we use the toast.promise function to display a loading message while the registration request is being processed, and a success message for 5 seconds after the request succeeds. If the request fails, we display an error message using the toast.error function.

Note that the toast.promise function expects a Promise as its first argument. In this example, we pass it a resolved Promise with the string "Registration succeeded" as the value. You can replace this with an actual Promise that resolves when the registration request is complete.
